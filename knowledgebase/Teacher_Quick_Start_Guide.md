Teacher Quick Start Guide

1. Sign in

- Open the Teacher Dashboard.
- Sign in with your teacher email and password.
- Use Forgot Password if you cannot sign in.

2. Check your classes

- Open Classes to see assigned groups.
- Select a class to see students, assigned courses, and pending reviews.
- Open a student detail view when you need more context.

3. Take attendance

- Open Attendance.
- Choose the class and date.
- Mark each student as Present, Absent, Late, or Excused.
- Add a short note only when useful.
- Click Save Attendance.

4. Find students who need help

- Open Needs Help.
- Check students flagged for pending review, low/no progress, or no recent activity.
- Open Detail to see courses, submissions, and attendance summary.

5. Review assignments

- Open Reviews.
- Filter by class, course, module, status, or student.
- Open uploaded proof when available.
- Read the student note.
- Write clear teacher feedback.
- Choose Complete, Needs Work, or Incomplete.

6. What students see

- Complete means the task is done.
- Needs Work asks the student to read feedback, improve, and resubmit.
- Incomplete asks the student to finish the task and resubmit.

7. Launch-week habit

- Start class by taking attendance.
- Check Needs Help before students begin practice.
- Review a small batch of submissions each day so feedback stays fresh.
